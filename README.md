# Ethereum-MarketPlace
This Dapp allows you to transact Ethereum around the globe. 


### Test Locally: 
```bash
cd client

npm run dev

```

Note: Update hardhat.config.js in smart_contracts/

1. Alchemy API Key
2. Your Private Key from metamask

### Build the app for deployment

```bash
npm run build

```
